# MovieBuzz

MovieBuzz is a simple frontend React project for searching and browsing movies from a local JSON database.

## Features

- Search movies by title.
- Filter movies by genre.
- View movie cards with poster, genre, year, duration, rating, and description.
- Open a details page for each movie.
- Uses `src/data/movies.json` as the local database.

## Run Locally

```bash
npm install
npm run dev
```
